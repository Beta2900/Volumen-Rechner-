# Volumen-Rechner

Ein einfaches Python-Programm zur Berechnung von **Volumen** und **Oberflächen** verschiedener geometrischer Körper.

## Funktionen

* Auswahl zwischen verschiedenen Körpern (z. B. Kugel)
* Berechnung von:

  * Volumen
  * Oberfläche
* Benutzerführung über ein Menü im Terminal

## Nutzung

1. Python-Datei in einer IDE öffnen (z. B. Thonny oder Visual Studio Code)
2. Programm starten
3. Im Hauptmenü einen Körper auswählen (z. B. Eingabe `1` für Kugel)
4. Im Untermenü auswählen:

   * Volumen oder Oberfläche
5. Geforderte Maße eingeben (nur Zahlen, ohne Einheiten)

## Technologien

* Python (Grundlagen: Funktionen, Bedingungen, Benutzereingaben)

## Lernziel

Dieses Projekt dient dazu, grundlegende Programmierkonzepte in Python zu üben, insbesondere:

* Strukturierung von Programmen
* Arbeiten mit Funktionen
* Benutzerinteraktion über die Konsole
